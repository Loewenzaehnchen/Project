# Ecotourism :earth_americas:
General description: Gives most suitable areas to visit in Central america + Mexico based on a filter system filled out by the user. Firstly, the user will be asked questions about themselves to establish what kind of person they are. Next, questions will be used the determine theie eco preference in the surrounding areas, including but not exclusively; nature, culture and destination.
